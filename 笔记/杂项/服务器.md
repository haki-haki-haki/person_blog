# 服务器

## 服务器简介

1. 服务器本质：一台联网的计算机

   服务器有公网IP（全球互联网唯一 IP 地址）

- 公网IP 全球唯一性，互联网设备可直接主动连接服务器

- 内网IP 运营商多层转发，不可被连接

  网站需要服务器，任何设备需要主动找到，需要公网IP。

2. 服务器随时运行

   网站需要随时被访问，正常情况下服务器一直运行。

3. 服务器Linux系统主流

### SSH连接服务器

终端命令SSH

```powershell
ssh 用户域名@服务器公网IP
## 例如 ssh ubuntu@123.45.67.89
```

第一次输入中断会询问是否新人这台机器，回复yes

接着输入密码，回车。

登陆成功显示

```powershell
ubuntu@your-server:~$
```



### 用apt安装Nginx

需要让服务器提供网页服务，需要软件Nginx

#### Nginx

- 持续坚挺80端口
- 游泳裤请求80端口，返回相应的应答

```powershell
sudo apt update
sudo apt install nginx -y
```

- `sudo`：以更高权限执行命令（类似 Windows 上的“以管理员身份运行”）
- `apt`：Ubuntu 上的软件安装工具

### 用IP地址访问默认界面

需要提前确认80端口的放行

打开浏览器，输入 `http://你的服务器公网IP`
