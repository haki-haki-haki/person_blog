# Markdown 语法学习笔记

## 一、标题语法

**作用：分级标题，自动排版**

```md
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

---

## 二、基础文本样式

**作用：重点标记文字**

```md
**加粗文字**
*斜体文字*
~~删除线文字~~
==高亮文字==
```

效果：**加粗文字**、*斜体文字*、~~删除线文字~~、

## 三、引用语法 \&amp; 嵌套引用（重点）

**作用：摘抄、备注、对话回复、分层注释**

#### 1\. 单层引用

```md
> 这是一级引用内容
```

> 这是一级引用内容
> 
> 

#### 2\. 多层嵌套引用（超常用）

```md
> 原文：BSP 是硬件底层驱动层
>> 解释：只操作硬件，不写业务逻辑
>>> 规则：上层调用下层，禁止下层调用上层
```

> BSP 是硬件底层驱动层
> 
> 只操作硬件，不写业务逻辑
> 
> 上层调用下层，禁止下层调用上层
> 
>

## 四、分割线 \-\-\- / \*\*\*

**作用：分割区块、区分知识点、页面整洁**

```md
内容1

---

内容2

```

## 十、字体颜色、大小、样式

**原生 Markdown 没有颜色字体，Typora 支持 HTML 样式**

#### 1\. 彩色字体

```md
<span style="color:red">红色重点文字</span>
<span style="color:#0099ff">浅蓝色文字</span>
```

#### 2\. 字体大小

```md
<span style="font-size:18px">18px 大号字体</span>
<span style="font-size:14px;color:#666">灰色小字注释</span>
```

#### 3\. 高亮背景

```md
<span style="background-color:#ffff99">黄色高亮重点内容</span>
```

#### 4\. 居中文字

```md
<div align="center">居中的文字内容</div>
```
